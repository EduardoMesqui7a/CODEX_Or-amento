# Roadmap MVP (8 semanas)

## Semana 1-2
- Extrair e testar núcleo Python (`core`).
- Garantir paridade funcional com fluxo legado.

## Semana 3-4
- Implementar API v1 de upload, inspect, jobs e resultado.
- Integrar worker assíncrono e persistência de jobs.

## Semana 5-6
- Construir frontend SaaS com landing, dashboard, processamento e histórico.
- Integrar chamadas reais da API e feedback de progresso.

## Semana 7
- Integrar autenticação completa (Clerk + validação JWT no backend).
- Isolar dados por usuário em todas as rotas.

## Semana 8
- Hardening, observabilidade, validação de limites e deploy.
- Checklist de go-live e runbook operacional.
